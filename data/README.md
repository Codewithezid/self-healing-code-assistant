# Data Directory

This folder is reserved for local datasets and logs.

## Layout

- `runtime/`
  - Local failure logs generated by the assistant when file logging is enabled.
- `finetune/`
  - Prepared training and validation JSONL files for Mistral fine-tuning.

These generated files are ignored by git.

Hosted deployments can avoid local runtime writes by setting `CODE_ASSISTANT_LOG_DESTINATION=upstash`.
